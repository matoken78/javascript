# 構築手順
1. Node.js のインストール
    1. 下記よりダウンロードしてインストール  
       https://nodejs.org/ja
    1. コマンドプロンプトを開き、下記を実行する  
       `npm install http-server@0.9.0 -g`

# 実行手順
1. http-server の起動
    1. index.html が存在するフォルダに移動する
    1. コマンドプロンプトを開き、下記を実行する  
       `http-server`
1. トップページにアクセス
    1. 下記を開く  
       http://localhost:8080/
